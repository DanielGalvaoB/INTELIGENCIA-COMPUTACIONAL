# Classificação do Dataset Iris usando Aprendizagem por Reforço

Este repositório contém a implementação de um trabalho prático que aplica o algoritmo Q-learning (Aprendizagem por Reforço) para resolver o problema de classificação do dataset Iris. O problema foi modelado como um ambiente de decisão sequencial.

## Arquivos Principais
* `codigo.py`: Implementação do ambiente, do agente Q-learning e do loop de treinamento/teste.
* `relatorio.pdf`: Documento detalhando a fundamentação teórica, metodologia, resultados e discussão do experimento.
* `graficos.png`: Resultados visuais das 30 rodadas independentes.
